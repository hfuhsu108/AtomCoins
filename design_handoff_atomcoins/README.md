# Handoff: 原子記帳 AtomCoins

## Overview

「原子記帳 AtomCoins」是一款個人財務管理應用程式，支援日常收支記帳、帳戶管理、股票交易紀錄、電子發票載具歸帳、以及財務報表。應用程式同時支援**手機版（390px）**與**桌面版（≥1024px）**兩種響應式佈局。

## About the Design Files

`designs/` 資料夾中的檔案是**以 HTML 製作的設計參考原型**——展示預期的視覺外觀與互動行為，**並非可直接使用的生產程式碼**。開發任務是在目標程式碼庫的既有環境（React、Vue、Flutter、SwiftUI 等）中，依據這些設計原型**重新實作 UI**，使用該環境已建立的模式與元件庫。若尚未有技術框架，請選擇最適合此專案的框架進行實作。

## Fidelity

這些設計為**高保真度（High-fidelity）**原型，包含最終的色彩、字體排版、間距與互動行為。開發者應盡可能**像素級還原**這些 UI，使用程式碼庫的既有元件庫與模式。

## Interactions & Behavior

### 全域互動
- **金額隱藏切換**：點擊眼睛圖示，所有金額替換為 `••••••`（6 字元）或 `••••`（4 字元），再點恢復
- **帳戶群組展開/收合**：chevron-right（收合）↔ chevron-down（展開），個別獨立控制

### 記帳畫面
- **類型切換**：Pill 按鈕，切換後整個表單內容變化
- **計算機**：支援連續運算（+ − × ÷）、百分比（%）、AC 清除、⌫ 退格、= 等號求值；運算中顯示運算式
- **分類選擇**：點擊分類 chip → 開啟 picker → 選擇主分類後右側更新子分類 → 選中子分類後自動關閉
- **進階展開**：點擊「進階」chip 展開備註/標籤/專案欄位
- **股票買賣切換**：兩個並排按鈕，選中時品牌色邊框 + `#EEF2FF` 背景

### 明細畫面
- **Tab 切換**：帳本/股票/發票載具
- **列表/日曆視圖切換**
- **篩選 Chip**：可點擊帶下拉選單（設計中僅展示 UI，未實作篩選邏輯）
- **桌面版選中效果**：左邊框 3px 品牌色 + 淡藍背景，右側顯示詳情
- **發票左滑**（手機版）：露出歸帳（綠色）和略過（灰色）操作按鈕

### 報表畫面
- **支出/收入切換**：切換後圓餅圖、排名、統計數據全部更新
- **較上月增減**：自動計算百分比，支出減少為好（綠色）、增加為壞（紅色），收入相反

### 設定畫面
- **Toggle 開關**：44×26px，啟用時滑塊右移 + 背景變品牌色，帶 transition 動畫 0.2s

### 動畫/過渡
- Toggle 開關：`transition: background .2s` + `transition: left .2s`
- 發票左滑：`transition: transform .2s`
- 設定項 hover（桌面）：背景變 `#FAFBFC`

## State Management

### 全域狀態
- `hidden: boolean` — 金額是否隱藏

### 首頁
- `comp: boolean` — 資產組成是否展開
- `groups: { cash, bank, card, stock: boolean }` — 各帳戶群組展開狀態

### 記帳
- `type: 'expense' | 'income' | 'transfer' | 'receivable' | 'payable' | 'stock'` — 當前交易類型
- `amount: string` — 輸入的金額運算式
- `cat: string` — 選中的主分類 ID
- `sub: string | null` — 選中的子分類 ID
- `catOpen: boolean` — 分類選擇器是否開啟
- `pickCat: string` — 選擇器中目前瀏覽的主分類
- `side: 'buy' | 'sell'` — 股票買賣方向
- `adv: boolean` — 進階欄位是否展開

### 明細
- `tab: 'ledger' | 'stock' | 'invoice'` — 當前子模組
- `view: 'list' | 'calendar'` — 帳本視圖模式
- `stockTab: 'holdings' | 'trades' | 'realized'` — 股票子 Tab
- `invoiceTab: 'unbooked' | 'booked'` — 發票子 Tab
- `selLedger / selHolding / selInvoice: string` — 桌面版選中項目 ID

### 報表
- `tab: 'expense' | 'income'` — 支出/收入切換

### 設定
- `defaultHide: boolean` — 預設隱藏金額開關

## Prop-based Customization

部分畫面支援 props 自訂（在設計原型中可透過 Tweaks 面板調整）：

- **`brand`** (color, default `#3B5BDB`)：品牌主色，影響選中狀態、按鈕、FAB、導覽等所有品牌色元素
- **`colorblind`** (boolean, default `false`)：色弱友善模式，綠色改用 `#1971C2`（藍色系），紅色改用 `#E8590C`（橘色系）
