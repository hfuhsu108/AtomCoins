# 04 — UI / 畫面設計

> 本檔為結構與內容規格，供與設計稿對齊。外觀（HTML/CSS 風格）由 Claude Design 另行產出。

## 全域設計系統

- mobile-first ＋ RWD：手機底部導覽列；**桌面（≥1024px）**改左側導覽欄＋雙欄內容。
- **底部導覽 5 鍵**（由左到右）：首頁、明細、中間突起圓形「＋」主按鈕、報表、設定。＋ 視覺最醒目。
- 列表為主、卡片用於摘要區；資訊密度優先、留白克制。
- Noto Sans TC、Font Awesome 6、**支出琥珀 `#F08C00` / 收入品牌藍 `#3B5BDB`**（色弱友善已內建於此配色）、**台股買紅 `#E03131` / 賣綠 `#2F9E44`**（依台灣慣例）、深/淺主題（以淺為主）。
- 金額千分位＋`NT$`＋正負上色；全域「眼睛」一鍵隱藏所有金額。
- **分類選擇器**：左右兩欄列表（左側母分類＋右側子分類），手機為 Bottom Sheet、桌面為 Dropdown。

## 畫面 1：首頁 Home

頂部摘要卡：**淨資產**大數字＋眼睛切換；下方小字「較上月 ±NT$X」；點擊展開組成（現金＋銀行＋投資市值 −信用卡未繳 ＋應收 −應付）。本月收支：收入(品牌藍)/支出(琥珀)並排＋結餘。帳戶列表：依類型分組(現金/銀行/信用卡/證券)可摺疊，每列 FA6 圖示＋名稱＋餘額；信用卡顯示「已用/額度」、證券顯示「持股市值」＋小字交割銀行餘額。右上：通知鈴鐺、隱藏金額。桌面：左側導覽欄；主內容左欄摘要、右欄帳戶列表。

## 畫面 2：記帳 Add（＋，**一打開即全螢幕**）

頂部 ×／儲存。最上類型列(可橫捲)：支出·收入·轉帳·應收·應付·股票，預設支出。
- 支出/收入：上半大字金額、下半大數字鍵盤；類別(母→子，FA6 圖示網格，常用優先)；帳戶(預設主帳戶)；日期(預設今天)；可展開：備註/標籤/專案；底部動作：**拆帳**(多列)、**代墊**(某列標記代墊→自動轉應收)、進階(手續費/折扣、入帳日)；頂部範本快捷。
- 轉帳：轉出/轉入帳戶、金額、手續費、日期。
- 應收/應付：對象、金額、帳戶、日期、備註。
- 股票：買/賣、證券帳戶、代號(自動帶名)、股數、成交價、券商(自動算費)、交割銀行、成交日(自動推 T+2)；即時顯示交割金額與「交割銀行夠不夠」，不足警告仍可存。
- 桌面：置中 modal 或右側面板。

## 畫面 3：明細 Transactions

分頁切換：帳本 ｜ 股票 ｜ 發票載具。
- 帳本：月份切換、搜尋、篩選(帳戶/類別/標籤/專案/對象/類型)、列表⇄月曆；依日期分組，header 顯示當日收支小計；每列 類別圖示＋名稱/備註＋帳戶＋金額(上色)；拆帳列標記「拆帳」（每個拆帳列各自成列顯示）、轉帳用 A→B、應收/應付有結清狀態。
- 股票：投資總覽(總市值/未實現損益/報酬率)；次分頁 持股｜交易紀錄｜已實現損益；持股依證券帳戶分組顯示代號/名稱/股數/現價/平均成本/市值/未實現損益/報酬率。股票買/賣標籤依**台股慣例**（紅=買進、綠=賣出）。
- 發票載具：從財政部載具匯入、手動新增；發票匣(未歸帳)列表商家/金額/日期，每張左滑「歸帳/略過」；切換 未歸帳/已歸帳。
- 桌面：雙欄(左列表右詳情)。

## 畫面 4：報表 Reports

月份導航＋支出/收入切換。統計摘要（本月金額/日均/筆數）；分類圓餅圖＋排名列表；近 6 個月收支柱狀趨勢圖。桌面：雙欄。進階報表（多維度、投資專屬、淨資產趨勢、自訂報表）保留給後續階段。

## 畫面 5：設定 Settings

分組列表：個人資料(Google 登入)、備份與同步(Drive 備份/還原、**自動備份開關**、上次備份時間、CSV 匯入/匯出)、帳戶管理(含信用卡出帳/繳費/額度/自動扣繳、證券交割銀行/券商)、分類設定(母/子＋FA6 icon＋色＋排序，支出/收入兩套)、標籤/專案/對象管理、券商設定(折數/最低費/捨去)、範本管理、偏好(主題、收支配色含色弱友善、金額隱藏預設)、通知(繳費提醒、提醒入帳)、關於。桌面：雙欄。

## Font Awesome 6 起手 icon 對應

導覽：首頁`fa-house`、明細`fa-receipt`、新增`fa-plus`、報表`fa-chart-pie`、設定`fa-gear`。
帳戶：現金`fa-wallet`、銀行`fa-building-columns`、信用卡`fa-credit-card`、證券`fa-chart-line`。
分類示例：餐飲`fa-utensils`、交通`fa-car`、購物`fa-bag-shopping`、居住`fa-house-chimney`、醫療`fa-kit-medical`、娛樂`fa-gamepad`、薪資`fa-money-check-dollar`。
動作：搜尋`fa-magnifying-glass`、篩選`fa-filter`、月曆`fa-calendar-days`、隱藏金額`fa-eye`/`fa-eye-slash`、掃描`fa-qrcode`、拆帳`fa-arrows-split-up-and-left`。
