# 更新日誌

本專案遵循 [語意化版本](https://semver.org/lang/zh-TW/)；發布流程＝bump `package.json` 版號 → 補本檔 → push（自動部署）→ 打 git tag。

## v1.1.0 — 2026-07-21

> 功能擴充計畫（docs/09）批次 1–6 ＋後續調整。**核心功能待登入實機驗收**（空資料冒煙與 build/lint 已通過）。

### 新增
- 明細全域搜尋：跨月，關鍵字／類型／分類／帳戶／日期區間／金額範圍組合
- 交易範本：記帳表單「存為範本」、記帳頁快速套用 chips、設定頁管理
- 商家維度：交易商家欄位＋商家別名（載具冗長公司名對應店名）＋商家排行
- 年度報表：收支頁月／年視角切換（年摘要、12 月柱狀、全年分類、商家排行）＋年度消費熱力圖
- 帳本日曆檢視：月曆格顯示當日支出，點日看當日交易
- 淨資產趨勢圖：每日快照累積，報表新增「資產」分頁（折線＋區間切換）
- 分類管理：自訂大／小分類、圖示、顏色、上下排序；分類色套用於選擇器、明細列、報表
- 帳戶自訂排序（上下箭頭）
- 證券帳戶可填期初持股（追蹤前已持有部位，只計市值成本、不扣交割銀行現金）

### 調整
- 設定頁改二層（選單 → 子區塊），避免設定項越加越長
- 發票：已歸帳可展開品項明細＋歸帳把明細帶入交易備註；手動新增的發票可編輯／刪除
- 帳戶可刪除（連同其交易一併刪除）

## v1.0.0 — 2026-07-20

首個正式版（1.0 前的開發歷程見 git tag `v0.7.0` 與 `docs/05-roadmap.md` 階段 0–7）。

### 新增
- 完整記帳核心：帳戶／收支／轉帳／應收應付、拆帳、代墊，記錄日與入帳日雙軌
- 信用卡：帳單切期、繳費、分期、額度；台股現股：買賣、持股損益、T+2 交割、GAS 每日收盤價
- 電子發票載具匣：本機爬蟲每日自動抓＋歸帳／略過／手動新增
- 收支報表（分類 Donut、近 6 個月趨勢）與投資報表
- 備份匯出：JSON 全量＋交易 CSV（拆帳逐列展開）
- 深淺主題三段切換（淺／深／跟隨系統）
- 通知中心：週期提醒＋信用卡繳費提醒＋交割餘額不足
- PWA：安裝、離線、長按捷徑（記一筆／發票匣）
- 版本顯示與 in-app 更新機制：開啟自動檢查，新版出橫幅、設定頁可手動檢查更新
